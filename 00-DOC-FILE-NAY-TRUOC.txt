║                    🐛 XỬ LÝ LỖI                                ║
╚════════════════════════════════════════════════════════════════╝

❌ Lỗi: "Connection refused"
   ✅ Kiểm tra: Server có đang chạy không?
   ✅ Chạy lại: python3 -m http.server 8080

❌ Lỗi: Ảnh không hiện
   ✅ Kiểm tra: Đường dẫn file ảnh
   ✅ Kiểm tra: Tên file có đúng không (phân biệt hoa/thường)

❌ Lỗi: Nhạc không chạy
   ✅ Kiểm tra: File mp3 có tồn tại không
   ✅ Kiểm tra: Định dạng file (phải là .mp3)
   ✅ Thử: Chuyển sang file nhạc khác

❌ Lỗi: Trang trắng
   ✅ Mở: F12 → Console → Xem lỗi
   ✅ Kiểm tra: Code có lỗi cú pháp không
